# Oracle Location

## Where is the Oracle?

The Oracle layer is reached after completing MY CHART DATA AND SUMMARY in the Tree of Life Reader.

Members must move through the MY CHART DATA AND SUMMARY stage first. Once that stage is complete, the main Reader area and Oracle layer become available.

## Credits for Oracle

Oracle-layer usage draws from the member's Oracle/app credits (shown as credits_remaining in the account). These are separate from Star Support chat credits.

## If Oracle is not appearing

- Confirm the member has completed MY CHART DATA AND SUMMARY.
- Confirm the member has sufficient Oracle/app credits remaining.
- If access still fails after these checks, Star Support can raise a support request.
