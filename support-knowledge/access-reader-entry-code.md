# Reader Entry Code

## What is the Reader Entry Code?

The Reader Entry Code is the access code used to enter Star Support and certain Stoic Qabalah products. It is provided to Paid Access members.

## How to use it

When prompted, enter the code exactly as supplied — codes are case-sensitive. The code is sent to the email address registered on your account.

## If the code fails

- Check the email address used matches your Paid Access account.
- Check spam or junk folders.
- Codes expire after 10 minutes — request a new one if needed.
- If the code continues to fail, Star Support can raise a support request for admin review.

## Wording rule

The correct wording is always READER ENTRY CODE. It is not described as a magic link or login link.
